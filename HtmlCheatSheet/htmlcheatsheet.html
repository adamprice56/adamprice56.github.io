Html Cheat Sheet

Tag Description Example of code
p A paragraph <p> This is a paragraph </p>
div Used to separate pages into sections to add background colours and formatting to a specific part of a page <div> <p> This paragraph is inside a div </p> </div>
h1, h2, h3, h4, h5, h6 A header of different sizes. (1 being the biggest)

ol, ul and li Ordered list (ol), unordered list (ul) and list item (li)
  1. Item 1
  2. Item 2
  • Item 1
  • Item 2
 <ul> <li> List item in unordered list </li> </ul>
hr, br Horizontal line / horizontal spacing (hr) and break for vertical spacing between objects also used for creating a new line within a tag e.g. <p>(br). Useful for spacing out items on a page or creating sections.
Horizontal Line (hr)
New line (br)
Another new line (br)
Another horizontal line (hr) 		<br> </br> or <hr> </hr>
a Anchor (a) is mostly used to create links on an object or text, surrounding something with this will allow for you to add a href to open a link upon clicking
Let's Google something! 		<a> <p> An example link </p> </a>
img

Img is used to insert... you guessed it, images into a page. This can be a random picture, logo or an animated .gif file

 <lmg src="yourimage.jpg">
span Span is used to apply formatting within another set of tags e.g. a paragraph This can be useful for applying CSS formatting to specific words or statements. Hello <span> World! </span>
strong Strong is the HTML equivalent to bold. Surrounding text in the Strong tag will make it become bold, this can be used within other tags e.g. p <p> This <strong> example </strong> shows how to use strong </p>
em Em is HTML's version of Italics. Used in the same way as Strong, it can surround any word or tags to make the text within them become Italic. <p> This is an <em> amazing </em> paragraph! </p>
table, th, tr and td Table is, of course, used to create a Table within a page. Th creates headers for columns, tr creates a row and td creates a cell. A good example is the table this is written in.
This can become complicated if not layed out nicely, remember to indent code for things like this to help spread things out. 		<table border= 1 width = 50%>
